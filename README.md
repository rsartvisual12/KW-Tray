# KW-Tray
Ein einfaches Tray-Tool, das die aktuelle Kalenderwoche anzeigt. Per Klick öffnet sich ein kompakter Kalender für einen schnellen Überblick über die umliegenden Wochen. Keine Installation! // A simple tray tool showing the current calendar week. One click opens a compact calendar for a quick overview of surrounding weeks. No installation!

VORAUSSETZUNGEN:
  Windows 10 oder 11
  
DOWNLOAD:
  1. KW-Tray.EXE herunterladen
  2. Ausführen (keine Installation notwendig)

FUNKTIONEN:
  - KW-Zahl als Icon im System Tray (unten rechts)
  - Klick: Kalender oeffnen/schliessen
  - Zeigt 1 Monat zurueck + heute + 1 Monat vor
  - Blaettern mit Zurueck/Vor-Buttons oder Mausrad
  - "Heute"-Button springt zum aktuellen Monat
  - Always-On-Top Pin-Mode
  - Kalenderwochen in jeder Zeile
  - Wochenenden farblich markiert
  - Bright-/ und Dark Mode
  - Rechtsklick auf Icon: Beenden

ZUM SYSTEMSTART HINZUFÜGEN:
  1. Rechtsklick auf KW-Tray.EXE -> Verknüpfung erstellen
  2. Win+R drücken -> "shell:startup" eingeben -> ENTER drücken
  3. Verknüpfung hierher kopieren
